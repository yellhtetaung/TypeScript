## What is TypeScript

A programming language to address **shortcomings** of **JavaScript**

### TypeScript Benefits

- Static typing
- Code completion
- Refactoring
- Shorthand notations

| Statically-Type  | Dynamically-Type           |
| ---------------- | -------------------------- |
| (C++, C#, Java)  | (JavaScript, Python, Ruby) |
| init number = 10 | let number = 10            |
| number = "a"; ❌ | number = "a";              |
|                  | Math.round(number);        |

## TypeScript

```typescript
let x: number = 10;
x = "a";
```
